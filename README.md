#Overview:
This repository contains a collection of real-time computer vision projects built using **YOLOv8** and **OpenCV**. 
It demonstrates a progression from basic object detection to some advanced multi-object tracking and specific use-case implementations like 
personal protective equipment (PPE) compliance and traffic monitoring.

#PROJECTS IN THE REPO:

##Project 1: YOLO Basics
A foundational project demonstrating how to load a pre-trained YOLOv8 model and run inference on static images and webcams.

##Project 2: Car Counter
A traffic monitoring system that detects, tracks, and counts vehicles (cars, trucks, buses, motorbikes) as they pass through a designated zone on a highway.

##Project 3: Safety Gear (PPE) Checker
An automated safety compliance tool designed for construction sites. 
It scans video feeds or images to detect whether workers are wearing required Personal Protective Equipment, such as hardhats and high-visibility vests.
